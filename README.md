# Excel Data Analysis Project

## Overview

This project focuses on data analysis using Microsoft Excel. It transforms raw data into meaningful insights using formulas, pivot tables, and visualizations.

You get a clear view of trends, patterns, and key metrics.

## Features

* Data cleaning and preprocessing
* Pivot tables for summary analysis
* Charts for visual representation
* Lookup functions for data mapping
* Structured and organized sheets

## Tools Used

* Microsoft Excel
* Pivot Tables and Pivot Charts
* Functions like VLOOKUP, IF, COUNT, SUM
* Data formatting and validation

## File Included

* `Anudip EXCEL Project.xlsx`
  Contains datasets, analysis sheets, and visualizations

## How to Use

1. Download the `.xlsx` file
2. Open it in Microsoft Excel
3. Navigate through sheets for different analysis
4. Interact with pivot tables and charts

## Analysis Highlights

* Category-wise expense analysis
* Item-level spending breakdown
* Frequency of transactions
* Highest and second-highest expense tracking
* Visual charts for better understanding

## Data Source

* Dataset is included within the Excel file
* No external connection required

## Project Objective

* Analyze structured data efficiently
* Use Excel tools to generate insights
* Present data in a simple and visual format

## Future Improvements

* Automate analysis using macros
* Add dashboard sheet for quick overview
* Improve visualization design
* Integrate with Power BI for advanced reporting
